ПРОГРАМНЫЙ КОМПЛЕКС ПРОСМОТРА ВАЛЮТ В РЕАЛЬНОМ ВРЕМЕНИ(в рамках тестового задания)
доступен по адресу
http://92.255.78.131:8000/

Состоит из файлов:
demon.js 	-  демон, по заданому таймауту читает входящие данные,выбирает нужные из заданного набора и пишет их в БД
start.js 	-  стартовый файл обращения, слушает 8000 и запускает public/index.html
settings.js	-  файл настроек проекта
server.js	- основной файл сервера. Слушает заданный в настройках порт,по коннекту начинает с заданной периодичностью читать данные(двумя запросами -первые текущие значения, второй -массив усредненных за все время).В этом же файле подготавливается массив на выдачу с пересборкой данных(самое сложное -перерасчет координат,чтобы нормально отображались)
/public  	-  собственно, сам тестовый сайт
	index.html  - индексный файл запуска, подтягивает к сeбе bootstrap5 и vue.js
	main.css 	- немного оформления
	vue.js		- обработчик на стороне клиента-нечего особенно интересного нет, функция клиент вебсокета из интересного
